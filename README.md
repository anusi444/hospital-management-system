## 🌟 Overview

A powerful **Hospital Management System** built using **Spring Boot, JSP, JPA, and Lombok**.
Designed to streamline hospital operations with **role-based access control**.

---

## ⚡ Features

### 👨‍⚕️ Admin Panel

* ➕ Add / Update / Delete Doctors
* 🧾 Manage Patients
* 📊 View system records

### 🧑‍💼 Receptionist Panel

* 📅 Book Appointments
* 🧍 Manage Patient Entries

### 🔐 Authentication

* Role-based Login System
* Secure Access Control

## ⚙️ Setup & Run

### 🔧 Requirements

* Java 17+
* Maven
* MySQL
* IDE (IntelliJ / Eclipse)

---

### 🗄️ Database Setup

Update `application.properties`:

```
spring.datasource.url=jdbc:mysql://localhost:3306/hospital_db
spring.datasource.username=root
spring.datasource.password=your_password
spring.jpa.hibernate.ddl-auto=update
```

---

### ▶️ Run Project

```
mvn spring-boot:run
```

OR run main class:

```
HospitalManagementSystemApplication.java
```

---

### 🌐 Open in Browser

```
http://localhost:8080
```
